Triangle Tracker

This is a programme designed by me for the essential part to allow you to insert three different values for the different sides of a triangle so that it gives you an answer on the type of triangle it makes.

The types of Triangle include the following:

  • Equilateral Triangle
  • Scalene Triangle
  • Isosceles Triangle

ABOUT

This is a webpage with a Triangle Tracker that is easy to use and navigate through so that the user will be allowed to enter three sides of a triangle and check whether the triangle is Equilateral,Isoscles,Scalene or not a triangle.However, It will not allow users to input non-numerical values.
